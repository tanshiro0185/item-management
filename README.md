# 商品管理システム

## ・概要
このシステムでは、店舗で扱う商品の管理を行うことができます。

商品の新規登録から編集、削除を行うことができます。

## ・主な機能

・ログイン、ログアウト機能

・商品一覧画面

・商品新規登録、編集、削除機能


PHP 8.2.12

MySQL 15.1

Laravel 10.13.5
